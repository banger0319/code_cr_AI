# Frontend Review Skill

- Check accessibility, semantics, keyboard navigation, and focus management.
- Watch for unnecessary re-renders, large client bundles, and expensive synchronous work.
- Review state management, cleanup of listeners, timers, and subscriptions.
- Pay attention to XSS, unsafe HTML, and client-side data exposure.
