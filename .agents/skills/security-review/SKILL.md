# Security Review Skill

- Check authz/authn, trust boundaries, secret handling, and sensitive logging.
- Watch for injection, XSS, SSRF, path traversal, and command execution risks.
- Review error handling for information leakage and unsafe fallback paths.
- Prefer the smallest safe fix and call out blocking issues clearly.
